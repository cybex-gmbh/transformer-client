# Cybex/Transformer

## Installation

To get the latest version, simply require the project using [Composer](https://getcomposer.org):

```bash
$ composer require cybex/transformer
```

## Transformations
width: e.g. w-150

height: e.g. h-150

format: e.g. f-jpg

Combinations: w-150+h-200+f-jpg
